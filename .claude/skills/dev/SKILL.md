---
name: dev
description: Starta utvecklingsservern för excel-app
allowed-tools: Bash
---

Kör `npm run dev` i `/workspaces/Hackathon/excel-app` i bakgrunden.

Bekräfta därefter att servern startade och på vilken URL den lyssnar (typiskt http://localhost:5173).

Påminn användaren om att öppna den URL:en i webbläsaren, och att servern körs tills de stoppar den med Ctrl+C i terminalen.
