allowed-tools: Bash
---

Kör `npm run build` i `/workspaces/Hackathon/excel-app`.

Rapportera:
- Om bygget lyckades: berätta hur många moduler som transformerades och filstorlekarna på dist-filerna.
- Om bygget misslyckades: visa exakt vilka fel som uppstod (filnamn, radnummer, felmeddelande) och föreslå en fix för varje fel.

Håll svaret kort och tydligt.
